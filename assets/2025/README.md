# 2025 Media Assets

Place your 2025 race photos and video here.

## Photos
Name files as: `photo-01.jpg`, `photo-02.jpg`, ... `photo-12.jpg` (or more)

Recommended:
- JPEG format
- At least 1200px wide for gallery display
- Square crop works best for gallery grid

## Video
For the promo video, either:
1. Upload to YouTube/Vimeo and update the embed URL in index.html
2. Or place an MP4 here named `promo.mp4`

To enable video embed, find this section in index.html:
```html
<div class="video-placeholder">
    ...
</div>
<!-- Uncomment and add video URL when ready:
<iframe src="YOUR_VIDEO_URL" frameborder="0" allowfullscreen></iframe>
-->
```

Replace with your embed code.
